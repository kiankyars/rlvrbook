# Preface {.unnumbered}

This repository holds the source manuscript for *Reinforcement Learning from Verifiable Rewards*. The book is intentionally verifier-first: it treats the verifier as the core design object and keeps optimizer details in the background unless they are needed to explain how verifier signal becomes capability.

## How To Read This Book

If you want the conceptual spine first, read chapters 1 through 4. If you care most about failure modes, read chapters 7 and 8 immediately after chapter 1. If you want concrete case studies, jump to chapters 9 and 10 once the verifier taxonomy is familiar.

## Current Scaffold

The manuscript is organized into five parts:

1. Foundations
2. Verifier Design
3. From Verifiers to Capability
4. Failure Modes
5. Domains and Frontiers

Each chapter file already contains a drafting skeleton so the manuscript can be written chapter-by-chapter without redesigning structure along the way.
